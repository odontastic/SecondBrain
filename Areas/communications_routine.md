# Area: Communications Routine

**Status:** Active  
**Standard to Maintain:** Handle all communications in one focused daily session to minimize distractions

---

## Routine Overview

**When:** Morning, after completing the ONE THING, when settled in  
**Duration:** 15-30 minutes (adjust based on volume)  
**Purpose:** Process all communications in one session to avoid constant interruptions throughout the day

---

## Daily Communications Routine

### Setup (Before Starting)
- [ ] Close distracting tabs/apps
- [ ] Set phone to Do Not Disturb (if possible)
- [ ] Close unnecessary applications
- [ ] Set timer if needed (to stay focused)

### Email Processing
- [ ] Open email client
- [ ] Process inbox using principles:
  - **Delete:** Spam, newsletters you won't read, outdated items
  - **Respond:** Quick replies (< 2 min) - do immediately
  - **File:** Important but no action needed - move to folders
  - **Task:** Needs action - add to task list, file email
  - **Defer:** Complex responses - schedule time, file for later
- [ ] Check spam/junk folder (quick scan)
- [ ] Archive or delete processed items

### Messages & Notifications
- [ ] Check text messages
- [ ] Respond to urgent messages
- [ ] Add non-urgent responses to task list if needed
- [ ] Process app notifications (if applicable)
- [ ] Clear notification badges

### Social Media (If Applicable)
- [ ] Check only if necessary for specific purpose
- [ ] Resist browsing/scrolling - be intentional
- [ ] Process notifications/messages only
- [ ] Close immediately when done

### Task Creation
- [ ] From communications, add any action items to task list
- [ ] Link to relevant projects/areas if applicable
- [ ] Set due dates if needed

### Wrap-up
- [ ] Close communication apps when done
- [ ] Return phone to normal settings
- [ ] Note any follow-up needed in Current Section
- [ ] **Move on to next priority** - don't get stuck in communications

---

## Principles

### Focus
- One session per day (unless emergency)
- Process everything in that session
- Avoid checking throughout the day

### Efficiency
- Quick decisions: delete, respond, file, or task
- Don't let emails sit in inbox as to-do list
- Use filters/rules to automate organization

### Boundaries
- Not a browsing session - process and move on
- Don't let communications derail priorities
- Finish routine and return to important work

---

## Integration with Daily Workflow

**Placement:** After completing ONE THING  
**Why:** 
- Already accomplished main priority
- Morning freshness for decision-making
- Settled into work mode
- Sets up rest of day without communication interruptions

**Time Block:** Log actual time taken to track efficiency

---

## Weekly Review

**Questions to Ask:**
- Am I keeping to one session per day?
- Is communications routine interrupting deep work?
- Am I processing efficiently or getting distracted?
- Should time block be adjusted?

**Adjustments:**
- If taking too long: be more aggressive with deletion
- If missing important items: improve filtering/rules
- If still checking throughout day: strengthen discipline

---

## Standards

- Communications processed daily in one session
- Inbox maintained (not accumulating)
- No constant checking throughout day
- Important items captured as tasks
- Distractions minimized
