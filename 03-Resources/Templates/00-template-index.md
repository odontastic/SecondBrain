# Template Library Index

## Overview
This library contains standardized templates for creating consistent, well-structured content across the LifeOS PARA system. Each template is designed to work seamlessly with the unified YAML frontmatter standard and PARA methodology.

## Template Categories

### Universal Templates
Templates that can be adapted for multiple use cases and content types.

#### Enhanced Note Template
- **File:** `01-note-template-enhanced.md`
- **Use for:** General notes, concepts, facts, and ideas
- **Features:** Unified frontmatter, standardized sections, cross-referencing

#### Learning Template
- **File:** `03-learning-template-enhanced.md`
- **Use for:** Educational content, courses, books, articles
- **Features:** Structured learning capture, source tracking, application planning

#### Creating Template
- **File:** `04-creating-template-enhanced.md`
- **Use for:** Creative projects, artistic work, content creation
- **Features:** Process documentation, inspiration tracking, results analysis

### PARA-Specific Templates
Templates designed for specific PARA categories.

#### Project Template
- **File:** `06-project-template.md`
- **Use for:** All actionable projects with deadlines
- **Features:** Progress tracking, milestone management, success metrics

#### Area Template
- **File:** `07-area-template.md`
- **Use for:** Ongoing responsibilities and life areas
- **Features:** Standards definition, KPI tracking, review scheduling

#### Routine Template
- **File:** `05-routine-template.md`
- **Use for:** Recurring activities and processes
- **Features:** Step-by-step process, troubleshooting, optimization

### Journal Templates
Templates for different types of journaling and reflection.

#### Daily Journal Template
- **File:** `02-daily-journal-template.md`
- **Use for:** Daily reflection, planning, and tracking
- **Features:** Morning/evening sections, mood/energy tracking, gratitude

#### Weekly Journal Template
- **File:** `08-journal-template-weekly.md`
- **Use for:** Weekly review, planning, and reflection
- **Features:** Week overview, goal progress, lessons learned

## Template Features

### Standardized Frontmatter
All templates include:
- **Complete metadata** following LifeOS v2.0 standards
- **Proper categorization** for PARA alignment
- **Rich tagging** for improved searchability
- **Cross-referencing** capabilities
- **Version tracking** and change management

### Structured Content
Each template provides:
- **Logical section organization** for easy content creation
- **Guided prompts** to ensure comprehensive documentation
- **Consistent formatting** across all content types
- **AI-friendly structure** for improved assistance

### Integration Support
Templates are designed to:
- **Link seamlessly** with other PARA components
- **Support bi-directional referencing** between files
- **Enable automated workflows** through consistent structure
- **Facilitate knowledge discovery** through standardized categorization

## Usage Guidelines

### Template Selection
1. **Assess content type** - Choose the most specific template available
2. **Consider PARA category** - Use PARA-specific templates when applicable
3. **Review template features** - Select based on required functionality
4. **Adapt as needed** - Customize within the established framework

### Template Customization
- **Maintain structure** - Keep core sections intact
- **Enhance fields** - Add category-specific content as needed
- **Update frontmatter** - Always include complete metadata
- **Test integration** - Ensure cross-references work properly

### Quality Assurance
- **Complete frontmatter** - No template should be used without full metadata
- **Consistent formatting** - Follow established markdown conventions
- **Proper categorization** - Ensure accurate PARA classification
- **Link validation** - Verify all cross-references work
