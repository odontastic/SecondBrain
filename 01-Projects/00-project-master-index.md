## Project Management Guidelines

### Adding New Projects
1. Create project file in appropriate subdirectory
2. Use standardized naming convention
3. Include complete YAML frontmatter
4. Update this index file
5. Add to relevant area connections

### Project Status Updates
- **Active**: Currently being worked on
- **On Hold**: Temporarily paused
- **Completed**: Successfully finished
- **Cancelled**: No longer relevant

### Review Schedule
- **Weekly**: Review active project status
- **Monthly**: Assess project priorities
- **Quarterly**: Archive completed projects
