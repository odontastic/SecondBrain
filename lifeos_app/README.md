# LifeOS Application

A comprehensive personal Life Operating System application built with Flask.

## Installation

1. Install Python dependencies:
```bash
pip install -r requirements.txt
```

2. Initialize the database:
```bash
python scripts/init_db.py
```

3. Run the application:
```bash
python app.py
```

4. Open your browser to: http://localhost:5000

## Structure

- `app/` - Application modules and logic
- `static/` - CSS, JavaScript, images
- `templates/` - HTML templates
- `data/` - SQLite database
- `scripts/` - Setup and utility scripts

## Features

- Dashboard with PARA/GTD overview
- Daily planning with kickstarters and faith goals
- Projects, Areas, Resources, Archives management
- GTD task management
- History and growth tracking
- Administration panel
